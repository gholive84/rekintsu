# Contexto do Site — Inunda IA
# Este arquivo é lido automaticamente pela IA ao editar páginas.
# Edite aqui para manter a IA sempre atualizada.

## Estrutura de arquivos PHP
Todas as páginas internas usam este padrão:
```php
<?php
$root         = dirname(__DIR__);
$page_title   = 'Título da Página';
$page_description = 'Descrição para SEO';
require_once $root . '/includes/head-page.php';
require_once $root . '/../site/includes/header.php'; // ou require header diretamente
?>
<!-- conteúdo HTML -->
<?php require_once $root . '/includes/footer.php'; ?>
```

## CSS carregado em todas as páginas (head-page.php)
Os seguintes arquivos CSS são incluídos automaticamente — use as classes definidas neles:
- /site/assets/css/reset.css
- /site/assets/css/variables.css  ← variáveis CSS globais
- /site/assets/css/global.css     ← tipografia, layout base
- /site/assets/css/components.css ← botões, cards, badges, forms
- /site/assets/css/sections.css   ← seções hero, services, etc.
- /site/assets/css/inner-pages.css ← estilos de páginas internas
- /site/assets/css/responsive.css ← breakpoints mobile

## CARROSSEL DE IMAGENS — USE SEMPRE ESTE PADRÃO COMPLETO

IMPORTANTE: Quando o usuário pedir um carrossel ou slider de imagens, copie EXATAMENTE este bloco completo. Não omita nenhuma parte, especialmente o `<script>` de inicialização no final.

```html
<!-- SWIPER CSS — colocar dentro do <head> ou logo antes do bloco -->
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/swiper@11/swiper-bundle.min.css">

<!-- ESTRUTURA DO CARROSSEL — use uma classe única por página, ex: swiper-fotos, swiper-clientes -->
<div class="swiper swiper-fotos" style="width:100%;max-width:1200px;margin:0 auto">
  <div class="swiper-wrapper">
    <div class="swiper-slide">
      <img src="/site/assets/img/foto1.jpg" alt="Imagem 1" style="width:100%;height:400px;object-fit:cover;border-radius:12px">
    </div>
    <div class="swiper-slide">
      <img src="/site/assets/img/foto2.jpg" alt="Imagem 2" style="width:100%;height:400px;object-fit:cover;border-radius:12px">
    </div>
    <div class="swiper-slide">
      <img src="/site/assets/img/foto3.jpg" alt="Imagem 3" style="width:100%;height:400px;object-fit:cover;border-radius:12px">
    </div>
  </div>
  <div class="swiper-pagination"></div>
  <div class="swiper-button-prev"></div>
  <div class="swiper-button-next"></div>
</div>

<!-- SWIPER JS + INICIALIZAÇÃO — obrigatório, colocar antes do </body> -->
<script src="https://cdn.jsdelivr.net/npm/swiper@11/swiper-bundle.min.js"></script>
<script>
new Swiper('.swiper-fotos', {
    loop: true,
    autoplay: { delay: 3500, disableOnInteraction: false },
    pagination: { el: '.swiper-fotos .swiper-pagination', clickable: true },
    navigation: {
        nextEl: '.swiper-fotos .swiper-button-next',
        prevEl: '.swiper-fotos .swiper-button-prev',
    },
    slidesPerView: 1,
    spaceBetween: 24,
    breakpoints: {
        768:  { slidesPerView: 2 },
        1024: { slidesPerView: 3 }
    }
});
</script>
```

ATENÇÃO: O `new Swiper(...)` NO FINAL É OBRIGATÓRIO — sem ele o HTML existe mas o carrossel não funciona.
Use uma classe única (ex: `.swiper-fotos`, `.swiper-clientes`, `.swiper-depoimentos`) para evitar conflito se houver mais de um na página.

### AOS (Animate on Scroll) — Animações ao rolar
```html
<link href="https://unpkg.com/aos@2.3.4/dist/aos.css" rel="stylesheet">
<!-- adicionar data-aos="fade-up" nos elementos -->
<script src="https://unpkg.com/aos@2.3.4/dist/aos.js"></script>
<script>AOS.init({ duration: 600, once: true });</script>
```
